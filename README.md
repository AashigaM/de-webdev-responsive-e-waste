## Project Overview

This project was developed for my **Dual Enrollment (DE) Web Page Development** course as a demonstration of **responsive web design principles**. Titled **"Tech Cycle: E-Waste & Sustainability,"** the website explores the growing issue of electronic waste while showcasing modern front-end development techniques. The project emphasizes creating layouts that adapt seamlessly across desktop, tablet, and mobile devices using responsive CSS methodologies and semantic HTML5 structure.

---

## Key Technical Skills Demonstrated

### Responsive Web Design
Designed layouts that automatically adapt to different screen sizes using **CSS Media Queries**, ensuring an optimized viewing experience across desktop, tablet, and mobile devices.

### CSS Grid Layout
Implemented a responsive **CSS Grid** system using `repeat(auto-fit, minmax())` to create flexible card layouts that automatically reorganize based on available screen width.

### Flexbox Layouts
Built responsive content sections using **Flexbox**, allowing images and text to display side-by-side on larger screens while stacking vertically on smaller devices for improved readability.

### Fluid Images
Applied responsive image techniques using `max-width: 100%` and `height: auto` so all images resize proportionally without overflowing their containers.

### Semantic HTML5 & User Experience
Structured the webpage using semantic HTML5 elements including `<header>`, `<nav>`, `<main>`, `<section>`, and `<article>`, along with smooth-scrolling navigation and anchor links to improve accessibility and user experience.

### Research Integration & Source Attribution
Combined responsive design with research-based educational content about electronic waste, including properly credited images and a dedicated Works Cited section referencing reputable sustainability resources.
